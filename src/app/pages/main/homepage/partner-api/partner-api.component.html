Partner Data Migration

Partners can migrate their students' data to the VAC system using any of the ways listed below:

  • Partner integrates VAC system’s API
  • VAC system integrates Partner’s API

VAC System integrates Institution’s API

This allows institutions to test their APIs on our sandbox application before moving to production environment.

  • Institutions must follow VAC system’s API standard which will be provided on our platform
  • Institutions can test their API on our sandbox environment.
  • On completion of testing, the institution can configure their APIs on the live environment.
