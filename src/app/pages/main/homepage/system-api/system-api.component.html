Partner Data Migration

Partners can migrate their students' data to the VAC system using any of the ways listed below:

  • Partner integrates VAC system’s API
  • VAC system integrates Partner’s API

Partner Integrates VAC system’s API

VAC system’s API will enable partners to connect to the VAC platform for the purpose of allowing partners to migrate their graduate data onto the platform.

  • An API would be made available, which partners would be able to test on their environment.
  • Partners can view API test results or response on our sandbox environment.
  • On completion of testing, the partner can request a live/production url.
